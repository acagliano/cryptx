asn1_obj_t *  objs,
size_t  iter_count 
)

Parses ASN.1 encoded data and returns metadata into an array of structs.

Note
This function is recursive for any element of constructed form.
For DER-formatted RSA public keys, you will need to call this function twice to unpack the modulus and exponent. The second time should be on the ASN1_BITSTRING that encodes the modulus and public exponent. See the asn1_decode demo for details.
Parameters
asn1_dataPointer to ASN.1-encoded data.
lenThe length of the encoded data.
objsPointer to an array of asn1_obj_t structs to fill with decoded data.
iter_countMaximum number of ASN.1 elements to process before returning.
Returns
The number of objects returned by the parser. Zero indicates an error.

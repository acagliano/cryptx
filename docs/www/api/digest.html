bool cryptx_digest_compare(const void *digest1, const void *digest2, size_t len)

Compare two digests or buffers.

Note

This is a constant-time implementation.

Parameters

  • digest1 – Pointer to first buffer to compare.

  • digest2 – Pointer to second buffer to compare.

  • len – Number of bytes to compare.

Returns

true if the buffers are equal, false if not equal.

bool cryptx_digest_tostring(const void *digest, size_t len, uint8_t *hexstr)

Convert a digest to its hexstring representation.

Note

hexstr must be at least (2 * len + 1) bytes large.

Parameters

  • digest – Pointer to a buffer or digest.

  • len – Byte length of digest.

  • hexstr – Buffer to write the output hex string to.

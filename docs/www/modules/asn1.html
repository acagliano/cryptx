asn1_error_t cryptx_asn1_decode(void *data_start, size_t data_len, uint8_t seek_to, uint8_t *element_tag, size_t *element_len, uint8_t **element_data)

Decodes the ASN.1 data at the given address.

Seeks to an element from the front of the data.

Note

ASN1_END_OF_FILE will be returned if seek_to is invalid.

Note

NULL may be passed for element_tag, element_len, and/or element_data if you do not need to return that particular bit of information.

Parameters

  • data_start – Pointer to a block of ASN.1-encoded data.

  • data_len – Length of ASN.1-encoded block.

  • seek_to – Number of ASN.1 elements to skip before returning one.

  • element_tag – Masked tag value of the returned element.

  • element_len – Length of the returned element.

  • element_data – Pointer to the data of the returned element.

Returns

An asn1_error_t indicating the status of the operation.

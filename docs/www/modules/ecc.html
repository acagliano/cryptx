ec_error_t cryptx_ec_keygen(uint8_t *privkey, uint8_t *pubkey)

Generates a pair of public/private keys for ECC over SECT233k1.

Note

Output public key is a point on the curve expressed as two 30-byte coordinates encoded in little endian byte order and padded with zeros (if needed). You may have to deserialize the key and then serialize it into a different format to use it with some encryption libraries.

Note

Parameters

  • privkey – Pointer to EC private key buffer.

  • pubkey – Pointer to EC public key buffer.

Returns

A random 29-byte EC private key and associated public key.
